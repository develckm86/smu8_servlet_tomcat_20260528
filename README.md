# Servlet vs JSP 수업

## 수업 목표

자바 웹앱서버인 톰캣으로 웹앱 작성하는 방법을 익힙니다.

## 수업 따라하는 방법

1. 이 저장소를 내려받습니다.
2. IntelliJ IDEA를 실행합니다.
3. 수업에 맞는 폴더를 IntelliJ IDEA에서 엽니다.
   - 예: `s02_servlet_vs_jsp`
4. IntelliJ IDEA가 Gradle 프로젝트를 불러올 때까지 기다립니다.
5. 프로젝트 내부에 포함된 톰캣 폴더를 사용해 실행합니다.
6. 실행 후 브라우저에서 웹앱 동작을 확인합니다.

## 실행 방식

이 수업에서는 별도로 설치한 외부 톰캣이 아니라, 프로젝트 내부 폴더에 포함된 톰캣을 사용합니다. 따라서 수업을 따라 할 때는 내려받은 프로젝트 폴더 구조를 유지한 상태로 IntelliJ IDEA에서 열어야 합니다.
